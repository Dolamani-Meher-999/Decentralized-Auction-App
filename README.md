# 🎨 Decentralized Auction DApp

A full-stack decentralized auction marketplace built with:
- **Blockchain**: Solidity + Hardhat
- **Frontend**: React + ethers.js v5
- **Backend**: Node.js + Express + MongoDB

## ✨ Features

- ✅ Create auction items (on-chain + off-chain storage)
- ✅ Place bids with MetaMask
- ✅ Real-time updates via contract events
- ✅ Multiple auctions running simultaneously
- ✅ Beautiful 3D UI with glassmorphism effects
